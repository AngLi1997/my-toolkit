# 基于Electorn的开发者小工具桌面应用

- Electron
- Vue3
- TypeScript

## 项目层级

```
my-toolkit/
├── .cursor/                  # Cursor编辑器配置目录
├── .git/                     # Git版本控制目录
├── node_modules/             # 依赖包目录
├── public/                   # 静态资源目录
│   ├── favicon.ico           # 网站图标
│   └── index.html            # HTML入口文件
├── src/                      # 源代码目录
│   ├── assets/               # 资源文件目录
│   │   └── logo.png          # Vue logo图片
│   ├── components/           # 组件目录
│   │   └── HelloWorld.vue    # 示例组件
│   ├── App.vue               # 根组件
│   └── main.js               # Vue应用入口文件
├── .gitignore                # Git忽略文件配置
├── babel.config.js           # Babel配置文件
├── jsconfig.json             # JavaScript配置文件
├── main.js                   # Electron主进程入口文件
├── package-lock.json         # 依赖包版本锁定文件
├── package.json              # 项目配置文件
├── README.md                 # 项目说明文档
└── vue.config.js             # Vue项目配置文件
```
